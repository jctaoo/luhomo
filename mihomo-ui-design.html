运行设置

管理 mihomo 内核的网络监听、代理模式与控制接口。

代理运行
对应 ProxyRunningArguments 的基础行为。
运行模式
规则匹配、全局代理或全局直连。
代理监听端口
客户端通过此端口访问代理服务。
允许局域网连接
允许同一网络内的设备使用此代理端口。
绑定地址
“*”监听所有地址；可填指定 IPv4 或 IPv6 地址。
内核与日志
控制协议栈与控制台输出详细程度。
IPv6 流量
允许内核接受及处理 IPv6 流量。
日志级别
日志仅输出到控制台与控制页面。
Web UI 预设
开启公开 HTTP API 后，可使用第三方 mihomo Web UI 管理内核。
公开 HTTP API
默认关闭；开启后会向局域网暴露控制接口。
API 端口
Web UI 通过此端口连接 mihomo 控制接口。
Web UI 预设
选择内置或常用的第三方管理界面。
公开 API 会让同一网络中的设备访问控制接口。仅在可信网络中开启，并建议配合访问密钥使用。

使用小贴士

关于 Mihomo Lite 你想知道的都在这里

代理服务已开启

关闭窗口或退出本软件后，代理服务仍在后台正常运行，不影响您的网络连接。

规则模式运行

国内网站和游戏直连不受影响，无需频繁开关本软件。

资源占用极低

后台运行时内存占用约 30–60 MB，CPU 占用趋近于零，无需担心性能负担。
